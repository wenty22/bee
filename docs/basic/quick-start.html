Notice

To use Bee, the React version should be in the range >=16.8 and <18, we recommend v17.

Installation

In your React project, install Bee by running the following:

npm i @node-real/bee @emotion/react@^11 @emotion/styled@^11 framer-motion@^6

To use bee-icons, just install as follow:

npm i @node-real/bee-icons
