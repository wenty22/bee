Props

colorShceme

Type:

success | danger | warning | NRpurple | BRblue

Default:

NRpurple

Description:

Scene colors, only effect when `variant="scene"`

size

Type:

sm | md | rg

Default:

rg

isDisabled

Type:

boolean

Default:

false

Description:

If true, the button will be disabled.

isFullWidth

Type:

boolean

Default:

false

Description:

If true, the button will take up the full width of its container.

isLoading

Type:

boolean

Default:

false

leftIcon

Type:

React.ReactElement

Description:

If added, the button will show an icon before the button's label.

rightIcon

Type:

React.ReactElement

Description:

If added, the button will show an icon after the button's label.

loadingText

Type:

string

Default:

Loading...

Description:

The label to show in the button when isLoading is true

variant

Type:

scene | brand | second | ghost | text | link

Default:

scene

type

Type:

submit | reset | button
