Show / Hide

The Show and Hide components are wrappers for other elements and components to show or hide them based on a media query.

Import

import { Show, Hide } from '@totejs/uikit';
  • Show: Show the children if the media query matches.
  • Hide: Hide the children if the media query matches.

Usage

Breakpoint Prop

Use the breakpoint prop to pass in a custom query.

<Show breakpoint="(max-width: 1000px)">
  <Box>This text appears only on screens 1000px and smaller.</Box>
</Show>

Above / Below

The above and below props receive the theme's breakpoint keys.

  • above is similar to "min-width"
  • below is similar to "max-width"
<>
  <Show above="md">
    <Box>This text appears at the "md" value screen width or greater.</Box>
  </Show>
  <Hide below="lg">
    <Box>This text hides at the "lg" value screen width and smaller.</Box>
  </Hide>
</>

Props

above

Type:

string

Description:

A value from the breakpoints section in the theme. Will render children from that breakpoint and above. Default breakpoint values: md, lg

below

Type:

string

Description:

A value from the breakpoints section in the theme. Will render children from that breakpoint and below. Default breakpoint values: md, lg

breakpoint

Type:

string

Description:

A custom css media query that determines when the children are rendered. Will render children if that query resolves to true.
